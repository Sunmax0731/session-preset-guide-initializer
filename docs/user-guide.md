# ユーザーガイド

## 基本操作

1. サンプル入力を確認する
2. `npm test` を実行する
3. CLIでレポートを生成する
4. FindingsとNext Actionsを確認する

## レポートの読み方

- Status が `公開前確認へ進行可` の場合、必須項目の不足はありません。
- `修正が必要` の場合、Findings の field と Next を確認してください。
- 完了率は必須項目の入力状況を示します。手動レビューの合格を自動保証するものではありません。

## ユーザーが記録するもの

- 手動テスト結果。
- リリース前に残した判断理由。
- 競合や公式基準と比べて不足している点。
