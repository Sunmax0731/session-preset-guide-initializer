## 手動テスト準備

1. このリリースの Assets から `session-preset-guide-initializer-docs.zip` をダウンロードする。
2. ZIPを任意の作業フォルダへ展開する。
3. 展開した `docs/installation-guide.md` を読み、必要なランタイム、ローカルサーバー、ホストアプリ、対象ファイルを確認する。
4. 展開した `docs/manual-test.md` の「手動テスト準備」に記載されたコマンド、作業ディレクトリ、URL、必要ファイルを準備する。
5. ソースから確認する場合は、リポジトリを clone し、タグ `v0.1.0-closed-alpha.1` を checkout する。

## 手動テスト手順

1. `docs/installation-guide.md` に従って導入する。
2. `docs/user-guide.md` に従って主要操作を1回通す。
3. `docs/manual-test.md` の各テストケースを上から順に実施し、期待結果と実結果を記録する。
4. `docs/strict-manual-test-addendum.md` の厳格QCDS追補チェックを実施する。
5. `docs/qcds-strict-evaluation.md` と `docs/qcds-strict-metrics.json` を確認し、評価根拠が `S+ / 100` として再現できることを確認する。
6. 失敗、手順不明点、UI/UX上の迷い、導入で詰まった箇所があれば、リポジトリのIssueへ記録する。
