# Security / Privacy チェックリスト

## 基本方針

このプロダクトは、外部サービスへの本番書き込みや破壊的操作を自動実行しない。ユーザーが判断する前に、不足、警告、次アクションを表示する。

## チェック項目

- [x] サンプルデータに秘密情報、トークン、private keyを含めない。
- [x] レポートは入力の必須不足と状態警告を示すだけで、外部環境を書き換えない。
- [x] ログやレポートに認証情報全文を保存しない方針を明記している。
- [x] 導入手順はローカル実行を基本とし、追加ツールは E:\DevEnv 配下を使う。
- [x] 手動テストで実ファイル、外部API、公開操作が必要な場合は事前準備と対象パスを明記する。
